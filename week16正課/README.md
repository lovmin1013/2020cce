利用elipse()畫出圓形(設定圓心、寬、高)，size()視窗大小，及background()背景顏色
```C
void setup(){ //設定，只做一次
  size(400,200);
} 
void draw(){ //畫，每秒60次
  background(#AF43CE);
  ellipse(50,50, 80,80);
}//畫圓   圓心   寬 高
```
